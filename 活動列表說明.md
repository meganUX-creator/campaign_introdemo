# 活動列表說明

## 操作提示
- **點擊「上傳」按鈕**：出現右側抽屜顯示配置預覽。
- **充值類型點擊**：查看最低充值金額與次數（僅充值類活動模板有效）。

## 篩選與功能按鈕
| 項目名稱 | 類型 | 說明 |
| :--- | :--- | :--- |
| **+ 新增活動** | 按鈕 (綠色) | 點擊進入建立新活動的編輯頁面。 |
| **刷新頁面** | 按鈕 (橘色) | 重新載入當前頁面資料。 |
| **狀態** | 下拉選單 | 依據活動的啟用狀態（如：開啟、關閉）進行篩選。 |
| **活動模板** | 下拉選單 | 依據使用的模板類型（如：砸金蛋、充值活動）進行篩選。 |
| **幣種** | 下拉選單 | 依據結算幣種（如：CNY、USD）進行篩選。 |
| **語系** | 下拉選單 | 依據前端顯示的語言（如：簡中、英文、越南文）進行篩選。 |
| **活動 ID** | 輸入框 | 輸入特定的活動 ID 進行精確查詢。 |
| **活動名稱** | 輸入框 | 輸入活動關鍵字進行模糊查詢。 |
| **日期區間** | 日期選擇器 | 選擇活動的起止時間範圍進行篩選。 |
| **重置** | 按鈕 (橘色) | 清空所有已輸入的篩選條件。 |
| **查詢** | 按鈕 (藍色) | 執行篩選動作並更新下方列表。 |

## 列表欄位說明
| 表頭名稱 | 說明 |
| :--- | :--- |
| **ID** | 活動的唯一識別編號（例如：1001, 1002）。 |
| **排序** | 活動在前端展示的優先順序。 |
| **狀態** | 顯示活動目前是否處於啟動中（綠色勾選）或關閉（黑色叉叉）。 |
| **模板類型** | 區分活動是「自定義」或是「系統」預設模板。 |
| **起止時間** | 活動開放的有效日期區間。 |
| **幣種** | 該活動適用的貨幣單位。 |
| **語系** | 該活動在前端支援展示的語言種類。 |
| **活動模版** | 指派給該活動的具體功能模板（如：砸金蛋）。 |
| **前台展示名稱** | 使用者在 App 或網頁前端看到的活動標題。 |
| **預覽圖片** | 活動的 Banner 或縮圖，點擊「上傳」可更改。 |
| **黑名單層級** | 限制特定層級的使用者無法參與此活動。 |
| **指定代理** | 綁定特定代理商（Agent）才能看到的活動。 |
| **到達 & 中獎人數** | 數據監控：上方數字可能為參與人數，下方數字為中獎人數。 |
| **提現稽核流水比例** | 設定參與活動後，領取獎金所需的打碼量（流水）倍數（如：200%）。 |
| **稽核不通過手續費比例** | 若使用者未達流水要求強制提現時，需扣除的手續費比例（如：2%, 5%）。 |
| **充值類型** | 活動觸發的儲值條件（如：筆筆充）。 |
| **最低充值金額** | 參與該活動所需單筆最低充值門檻。 |
| **最低充值次數** | 參與該活動所需累積的最低充值次數。 |

## 操作按鈕
| 項目名稱 | 說明 |
| :--- | :--- |
| **關閉** | 立即停止該活動在前端展示。 |
| **複製** | 以此活動為範本，快速建立一個內容相同的新活動。 |
| **編輯** | 進入編輯頁面修改上述所有參數欄位。 |
| **刪除** | 永久移除該活動紀錄（通常需二次確認）。 |
