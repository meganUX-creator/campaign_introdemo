5-1. 優惠碼設置

設置優惠碼的基本信息，包括優惠碼名稱、類型、有效期及領取次數限制。

設置框架

切換按鈕：包含：上一步、下一步與退出按鈕。

左側選單：當前步驟為【優惠碼設置】。

详细参数说明

编号 项目名称 内容说明
a.1 优惠码名称 设置该优惠码的内部管理名称。
a.2 优惠码类型 选择优惠码的发放形式（如：通用码、唯一码）。
a.3 有效时间 设置优惠码可被领取的起止时间。
a.4 领取限制 设置单个用户可领取最高次数。
a.5 展示名称 设置前端用户看到的优惠码名称。

5-1. Promo Code Settings

Set basic information for the promo code, including name, type, validity period, and claim limits.

Setup Framework

Switch Buttons: Includes Previous Step, Next Step, and Exit buttons.

Sidebar Menu: Current step is [Promo Code Settings].

Detailed Parameter Description

No. Item Name Description
a.1 Promo Name Set the internal management name for this promo code.
a.2 Promo Type Select emission format (e.g., Generic Code, Unique Code).
a.3 Valid Time Set the start and end time during which the code can be claimed.
a.4 Claim Limits Set the maximum number of times a single user can claim it.
a.5 Display Name Set the name of the promo code visible to frontend players.

5-1. Cài đặt mã khuyến mãi (Promo Code)

Thiết lập thông tin cơ bản cho mã khuyến mãi, bao gồm tên mã, loại mã, thời hạn hiệu lực và giới hạn số lần nhận.

Khung thiết lập

Nút chuyển đổi: Bao gồm các nút: Bước trước, Bước tiếp theo và Thoát.

Menu bên trái: Bước hiện tại là [Cài đặt mã khuyến mãi].

Mô tả thông số chi tiết

STT Tên mục Mô tả nội dung
a.1 Tên mã KM Thiết lập tên quản lý nội bộ cho mã khuyến mãi này.
a.2 Loại mã KM Chọn hình thức phát hành (Ví dụ: Mã chung, Mã duy nhất).
a.3 Thời gian hiệu lực Thiết lập thời gian bắt đầu và kết thúc có thể nhận mã.
a.4 Giới hạn nhận Thiết lập số lần tối đa một người dùng có thể nhận.
a.5 Tên hiển thị Thiết lập tên mã khuyến mãi mà người chơi thấy ở giao diện frontend.
