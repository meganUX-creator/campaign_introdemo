5-6. 提款条件

设置领取优惠码奖励后，提现需满足的流水倍数及计算规则。

详细参数说明

编号 项目名称 内容说明
1. 打码流水倍数 奖励金额需满足的流水倍数（如：1倍流水）。
2. 流水计入场馆 指定某些游戏场馆的投注流水才计入提款门槛。
3. 提款手续费 若未完成流水即提款，设置相应的手续费扣除比例。

5-6. Withdrawal Conditions

Set the turnover multiples and calculation rules required for withdrawal after claiming promo code rewards.

Detailed Parameter Description

No. Item Name Description
1. Turnover Multiplier The turnover multiple required for the reward amount (e.g., 1x turnover).
2. Eligible Venues Specify game venues whose bets count towards the withdrawal threshold.
3. Withdrawal Fee Set the fee deduction percentage if withdrawing without completing turnover.

5-6. Điều kiện rút tiền

Thiết lập bội số doanh thu và quy tắc tính toán cần đáp ứng để rút tiền sau khi nhận phần thưởng mã khuyến mãi.

Mô tả thông số chi tiết

STT Tên mục Mô tả nội dung
1. Bội số doanh thu cược Bội số doanh thu yêu cầu cho số tiền thưởng (Ví dụ: 1x doanh thu).
2. Sảnh tính doanh thu Chỉ định các sảnh trò chơi mà doanh thu cược được tính vào ngưỡng rút tiền.
3. Phí rút tiền Thiết lập tỷ lệ khấu trừ phí nếu rút tiền khi chưa hoàn thành doanh thu.
