优惠码兑换限制

设定全站的优惠码每日兑换限制，还有根据某些会员帐号做出不同兑换次数调整限制的功能。

功能说明

编号 说明内容
1. 设置用户每天可兑换优惠码多少次
2. 该区域可以看到特定用户的每日限制兑换次数，并且单独针对某些会员设置每日兑换次数。
  • a-1. 新增会员账号：点击后会出现每日兑换限制的弹窗。
  • a-2. 设定限制的会员跟次数：按下确认即新增完成。
  • a-3. 编辑限制：若对于某些会员账户编辑该会员的限制次数。

Redemption Limits

Set site-wide daily redemption limits for promo codes, and includes the function to adjust redemption limits based on certain member accounts.

Feature Description

No. Description
1. Set how many times a user can redeem promo codes per day
2. This area displays the daily redemption limit for specific users, and allows setting daily redemption limits for specific members individually.
  • a-1. Add Member Account: Clicking this will bring up the daily redemption limit prompt.
  • a-2. Set Member and Limit: Click confirm to complete the addition.
  • a-3. Edit Limits: If you need to edit the limit for certain member accounts.

Hạn chế đổi mã

Thiết lập giới hạn đổi mã KM hàng ngày trên toàn trang, cũng bao gồm chức năng điều chỉnh số lần đổi dựa trên các tài khoản thành viên nhất định.

Mô tả tính năng

STT Mô tả nội dung
1. Thiết lập số lần người dùng có thể đổi mã KM mỗi ngày
2. Khu vực này hiển thị giới hạn đổi mã hàng ngày cho người dùng cụ thể, và cho phép thiết lập giới hạn đổi mã hàng ngày cho từng thành viên cụ thể.
  • a-1. Thêm tài khoản thành viên: Nhấp vào đây sẽ hiển thị hộp thoại giới hạn đổi mã hàng ngày.
  • a-2. Đặt thành viên và giới hạn: Nhấn xác nhận để hoàn tất việc thêm.
  • a-3. Chỉnh sửa giới hạn: Nếu bạn cần chỉnh sửa giới hạn cho tài khoản thành viên nhất định.
