5-4. 风控设定

设置领取优惠码时的安全检测规则，防止恶意刷取。

详细参数说明

编号 项目名称 内容说明
1. 风险检测项目 包含同IP、同手机、同姓名、同E-mail等重复性检测。
2. 自动拦截阈值 当违反规则的次数达到设定值时，系统自动拒绝领取。

5-4. Risk Control Settings

Set security detection rules when claiming promo codes to prevent malicious exploitation.

Detailed Parameter Description

No. Item Name Description
1. Risk Detection Items Includes redundancy checks for Same IP, Phone, Name, E-mail, etc.
2. Auto-Block Threshold System automatically rejects claims when the number of violations reaches the set limit.

5-4. Thiết lập kiểm soát rủi ro

Thiết lập các quy tắc kiểm tra an ninh khi nhận mã khuyến mãi để ngăn chặn việc trục lợi ác ý.

Mô tả thông số chi tiết

STT Tên mục Mô tả nội dung
1. Hạng mục kiểm tra rủi ro Bao gồm các kiểm tra trùng lặp như trùng IP, số điện thoại, họ tên, E-mail, v.v.
2. Ngưỡng tự động chặn Hệ thống tự động từ chối nhận thưởng khi số lần vi phạm đạt đến giá trị thiết lập.
