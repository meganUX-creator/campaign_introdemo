5-3. 活动条件

设置兑换该优惠码所需满足的充值或其他行为条件。

详细参数说明

编号 项目名称 内容说明
1. 首充要求 是否要求用户必须是首次充值后才能兑换。
2. 最低充值金额 设置领取该码前需满足的单笔或累计最低充值金额。
3. 稽核方式 分为人工稽核或自动稽核兑换申请。

5-3. Activity Conditions

Set the deposit or other activity requirements that must be met to redeem the promo code.

Detailed Parameter Description

No. Item Name Description
1. First Deposit Req. Whether the user must have made their first deposit before redemption.
2. Min. Deposit Amount Set the single or cumulative minimum deposit required before claiming the code.
3. Audit Method Choose between manual or automatic audit for redemption applications.

5-3. Điều kiện hoạt động

Thiết lập các yêu cầu về nạp tiền hoặc hành vi khác cần đáp ứng để đổi mã khuyến mãi.

Mô tả thông số chi tiết

STT Tên mục Mô tả nội dung
1. Yêu cầu nạp đầu Yêu cầu người dùng phải thực hiện nạp tiền lần đầu mới có thể đổi mã.
2. Số tiền nạp tối thiểu Thiết lập số tiền nạp tối thiểu (theo đơn hoặc lũy kế) cần đáp ứng trước khi nhận mã.
3. Phương thức duyệt Gồm duyệt thủ công hoặc duyệt tự động yêu cầu đổi mã.
