救援金活动模板

当用户在特定游戏场馆产生亏损且余额低于设定阈值时提供的补偿性活动。

救援金活动后台配置界面
配置项 说明
亏损计算 计算公式：(总输赢 - 总优惠) 的绝对值。
余额限制 用户申请时，钱包总余额必须低于设定的金额（如 5 元）。

Rescue Fund Template

Compensatory activity provided when a user incurs losses in specific game venues and their balance falls below a set threshold.

Rescue Fund Activity Backend Configuration Interface
Config Item Description
Loss Calculation Formula: Absolute value of (Total Win/Loss - Total Promotion).
Balance Limit Total wallet balance must be lower than the set amount (e.g., 5) at the time of application.

Mẫu quỹ cứu trợ

Hoạt động bù đắp được cung cấp khi người dùng thua lỗ tại các sảnh trò chơi cụ thể và số dư giảm xuống dưới mức ngưỡng thiết lập.

Giao diện cấu hình Backend cho hoạt động Quỹ cứu trợ
Mục cấu hình Mô tả
Tính toán thua lỗ Công thức: Giá trị tuyệt đối của (Tổng Thắng/Thua - Tổng Khuyến mãi).
Giới hạn số dư Tổng số dư ví phải thấp hơn số tiền đã thiết lập (ví dụ: 5) tại thời điểm đăng ký.
