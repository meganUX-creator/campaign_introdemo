# 活动模板设置

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **活动模板** | |
| 1-1. | 选择模板 | 选择对应活动类型的模板。活动条件会出现对应的设置条件（例如：连续性活动无提款条件）。 |
| 1-2. | 模板类型 | 定义请参考下表 **1-2-1**。 |
| **1-2-1.** | **模板类型说明** | |
| a. | 系统有专题页 | 该模板有默认图片，并且也有在前端列表排序。 |
| b. | 系统无专题页 | 该模板没有系统自带的图片。需要在活动图片步骤上传及设定活动类别。 |
| c. | 自定义 | 仅会出现：活动设置、活动图片、参与对象与名单、提交预览。 |
| **2.** | **时间设定** | |
| 2-1. | 指定时间段 | |
| 2-1.1. | 展示时间 | 适用于活动预热的开始到结束的展示时间。 |
| 2-1.2. | 进行时间 | 活动开始的时间周期。 |
| 2-2. | 长期展示 | 活动没有结束时间。 |
| **3.** | **活动类型设置** | |
| 3-1. | 活动类型 | 决定该活动是否运作（开关）。 |
| 3-2. | 活动币种 | 设置其中一个活动币种。 |
| **4.** | **活动名称** | |
| 4-1. | 后台名称 | 在后台活动列表默认展示的名称，让运营容易辨识名称。 |
