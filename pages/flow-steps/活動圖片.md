# 活动图片

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **活动类型设置** | |
| 1-1. | 活动类型 | 设定在客户端的活动列表分类的类型（仅当选择“系统有专题页”时显示）。详情请参考前页 (1-2-1)。 |
| 1-2. | 活动语系 | 活动内容跟图片支持的展示语系。 |
| **2.** | **排序** | |
| 2-1. | Banner 排序 | 活动列表的 Banner 排序（数字越小越靠前）。 |
| **3.** | **语系设定** | |
| 3-1. | 语系复选 | 根据后台支持的语系进行复选，勾选后会出现对应的图片及内容设置。 |
| 3-2. | 活动标题 | 填写不同语系的活动标题，用于在前台显示。 |
| 3-3. | 语系 Tab 切换 | 切换不同的语系 Tab，上传不同的图片跟描述内容。 |
| **4.** | **活动图片上传** | |
| 4-1. | 图片素材上传 | 针对同一语系，分别在“PC 站内列表图片”、“H5 站内列表图片”、“APP 端列表图片”等不同位置上传对应尺寸的素材。 |
| 4-2. | 跳转类型 | 针对每一张图片设定“跳转类型”，决定用户点击该图片後的跳转行为（如：不跳转、跳转至特定活动等）。 |
| **5.** | **活动描述** | |
| 5-1. | 活动内容 | 使用富文本编辑器填写活动的详细描述、规则或介绍。 |
| **6.** | **备注** | |
| 6-1. | 备注内容 | 填写仅供后台运营人员查看的内部备注信息。 |
