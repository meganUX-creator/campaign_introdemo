# 4-7. 提款条件

## 简体中文 (SC)

奖励提现的稽核标准，包含稽核手续费比例、提款流水比例、流水计算基数（本金/礼金）以及稽核场馆设置。

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **稽核手续费** | 稽核不通过时的手续费比例（默认为 100%）。 |
| **2.** | **提款稽核流水** | 根据设定的比例，对提款礼金的有效流水进行审核。 |
| **3.** | **提现流水类型** | （仅限充值模板）选择计算基数：仅本金，或本金 + 礼金。 |
| **4.** | **稽核场馆** | 指定计入流水统计的游戏场馆（点击后在右侧抽屉设置）。 |

---

## English (EN)

Audit standards for reward withdrawals, including fee ratio, turnover ratio, calculation base (principal/bonus), and game venue settings.

| No. | Item Name | Description |
| :--- | :--- | :--- |
| **1.** | **Audit Fee** | Fee percentage if the audit fails (default 100%). |
| **2.** | **Withdrawal Audit** | Review turnover for the withdrawal bonus based on the specified ratio. |
| **3.** | **Turnover Type** | (Deposit templates only) Choose base: Principal Only, or Principal + Bonus. |
| **4.** | **Audit Venues** | Specify game venues for turnover calculation (set via right drawer). |

---

## Tiếng Việt (VN)

Tiêu chuẩn kiểm tra và yêu cầu bội số doanh thu để rút tiền thưởng, bao gồm tỷ lệ phí, tỷ lệ doanh thu rút tiền, cơ sở tính doanh thu (tiền gốc/thưởng) và thiết lập sảnh trò chơi.

| STT | Tên mục | Mô tả nội dung |
| :--- | :--- | :--- |
| **1.** | **Phí kiểm tra** | Tỷ lệ phí nếu kiểm tra không đạt (mặc định 100%). |
| **2.** | **Kiểm tra rút tiền** | Kiểm tra doanh thu cho tiền thưởng rút dựa trên tỷ lệ đã thiết lập. |
| **3.** | **Loại doanh thu** | (Chỉ mẫu nạp tiền) Chọn cơ sở: Chỉ tiền gốc, hoặc Tiền gốc + Tiền thưởng. |
| **4.** | **Sảnh kiểm tra** | Chỉ định các sảnh trò chơi được tính vào doanh thu (thiết lập qua ngăn kéo bên phải). |
