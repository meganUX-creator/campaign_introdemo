# 4-3. 参与对象与名单

## 简体中文 (SC)

没有设置参数代表全站的人都可以参加。

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **充值层级** | |
| 1-1. | 可参加充值层级 | 可选择对应多个充值层级。 |
| **2.** | **指定代理** | 选取维度从不指定到整条代理或者针对某些代理会员。 |
| **3.** | **新会员限制** | |
| 3-1. | 活动开始后注册的新会员 | 开启后，仅限活动开始后注册的新会员参加。 |
| **4.** | **指定会员等级** | 可设定为管理员或不同等级的会员。 |
| **5.** | **用户标签** | 用于筛选目标用户群体。 |
| **6.** | **黑名单设置** | |
| 6-1. | 黑名单限制类型 | 根据黑名单类型进行限制不可参加。 |
| 6-2. | 用户标签 | 用于筛选目标用户群体。 |

---

## English (EN)

If no parameters are set, it means everyone on the site can participate.

| No. | Item Name | Description |
| :--- | :--- | :--- |
| **1.** | **Deposit Tiers** | |
| 1-1. | Participating Tiers | Multiple deposit tiers can be selected. |
| **2.** | **Designated Agents** | Selection ranges from no designation to entire agent lines or specific agent members. |
| **3.** | **New Member Restriction** | |
| 3-1. | Post-Activity Registration | When enabled, only members who register after the activity starts can participate. |
| **4.** | **VIP Levels** | Can be set for administrators or different VIP levels of members. |
| **5.** | **User Tags** | Used to filter target user groups. |
| **6.** | **Blacklist Settings** | |
| 6-1. | Restriction Type | Restrict participation based on the type of blacklist. |
| 6-2. | User Tags | Used to filter target user groups for exclusion. |

## Tiếng Việt (VN)

Nếu không thiết lập tham số, tất cả mọi người trên trang web đều có thể tham gia.

| STT | Tên mục | Mô tả nội dung |
| :--- | :--- | :--- |
| **1.** | **Phân khúc nạp tiền** | |
| 1-1. | Phân khúc được tham gia | Có thể chọn nhiều phân khúc nạp tiền tương ứng. |
| **2.** | **Đại lý chỉ định** | Kích thước lựa chọn từ không chỉ định đến toàn bộ tuyến đại lý hoặc dành cho một số thành viên đại lý cụ thể. |
| **3.** | **Hạn chế thành viên mới** | |
| 3-1. | Đăng ký sau khi bắt đầu | Sau khi bật, chỉ giới hạn cho các thành viên mới đăng ký sau khi hoạt động bắt đầu tham gia. |
| **4.** | **Cấp độ VIP** | Có thể thiết lập là quản trị viên hoặc các cấp độ thành viên khác nhau. |
| **5.** | **Thanh thẻ người dùng** | Được sử dụng để lọc các nhóm người dùng mục tiêu. |
| **6.** | **Thiết lập danh sách đen** | |
| 6-1. | Loại hạn chế danh sách đen | Hạn chế không thể tham gia dựa trên loại danh sách đen. |
| 6-2. | Thanh thẻ người dùng | Được sử dụng để lọc các nhóm người dùng mục tiêu để loại trừ. |
