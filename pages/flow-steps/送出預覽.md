# 4-8. 送出预览

## 简体中文 (SC)

发布前的最后核对环节，通过全面检查配置信息并完成安全验证，确保活动配置准确发布。

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **配置核对** | 在预览页面全面检查各项活动参数设置。 |
| **2.** | **确认送出** | 确认无误后点击右上角“确认送出”按钮。 |
| **3.** | **安全验证** | 输入对应的安全码/验证信息以完成最终配置发布。 |

---

## English (EN)

Final check-over before publishing, including a comprehensive configuration review and security authentication to ensure accurate release.

| No. | Item Name | Description |
| :--- | :--- | :--- |
| **1.** | **Config Check** | Comprehensive review of all set parameters on the preview page. |
| **2.** | **Submit** | Click the "Submit" button in the upper right corner after confirmation. |
| **3.** | **Security Auth** | Enter the verification code to complete the final configuration publish. |

---

## Tiếng Việt (VN)

Bước kiểm tra cuối cùng trước khi xuất bản, bao gồm xem xét toàn diện cấu hình và xác thực bảo mật để đảm bảo phát hành chính xác.

| STT | Tên mục | Mô tả nội dung |
| :--- | :--- | :--- |
| **1.** | **Kiểm tra cấu hình** | Kiểm tra toàn diện các thông số cài đặt trên trang xem trước. |
| **2.** | **Xác nhận gửi** | Nhấn nút "Xác nhận gửi" ở góc trên bên phải sau khi kiểm tra xong. |
| **3.** | **Xác thực bảo mật** | Nhập mã xác thực/thông tin bảo mật để hoàn tất xuất bản cấu hình. |
