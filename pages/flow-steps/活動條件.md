# 4-4. 活动参与条件

## 简体中文 (SC)

配置活动的参与周期（如：首充、每日充）、充值渠道、审核方式，以及现金或券类奖励的触发条件。

| 編號 | 項目名稱 | 內容說明 |
| :--- | :--- | :--- |
| **1.** | **周期** | 根据不同的活动充值频率进行选择（如：笔笔充、首充、每日充值等）。 |
| 1-1. | 充值类别选择 | 在充值页面的分类选择。 |
| 1-2. | 充值类别管理 | 点击会出现弹窗（详情请参见下页）。 |
| **2.** | **充值渠道** | 选择可充值的多个指定渠道。未指定默认全部充值渠道。 |
| **3.** | **审核方式** | **人工审核：**选中后，所有符合条件的申请均需经后台管理员手动核准后方可发放奖励。<br>**自动审核：**选中后，系统将根据预设规则自动触发奖励发放，无需人工干预。<br>**金额门槛设定：**此处会根据选取的币种，在输入框中填写具体金额。若设定该值，当奖励金额超过此数字时，系统将强制转为人工审核。<br>*\*注意：填写 0 或留空可能代表所有金额均需经过手动审核（具体视系统底层逻辑而定，建议初次设定时进行测试）。* |
| **4.** | **排序** | |
| 4-1. | 充值页面排序 | 充值页面的排序设置（详情请参见下页）。 |
| **5.** | **现金奖励** | |
| 5-1. | 发放模式 | 选择其一。该选项会影响触发条件的奖励类型。 |
| 5-2. | 数值设定 | 选择其一。该选项会影响触发条件的奖励类型。 |
| **6.** | **券类奖励** | 免费跟真人下注券两者其一。 |
| **7.** | **触发条件** | 根据现金奖励跟券奖励的设置有不同组合。点击“新增条件”可设置多组。 |

## English (EN)

Configure activity participation cycles (e.g., first deposit, daily), deposit channels, audit methods, and trigger conditions for cash or coupon rewards.

| No. | Item Name | Description |
| :--- | :--- | :--- |
| **1.** | **Period** | Select based on deposit frequency (e.g., Every Deposit, First Deposit, Daily, Weekly). |
| 1-1. | Category Selection | Select category on the deposit page. |
| 1-2. | Category Management | Click to open a pop-up (refer to the next page for details). |
| **2.** | **Deposit Channels** | Select multiple designated channels. Defaults to all channels if unassigned. |
| **3.** | **Audit Method** | **Manual Approval:** When selected, all eligible applications require manual approval by an admin before rewards are issued.<br>**Auto Approval:** When selected, the system automatically triggers reward issuance based on preset rules without manual intervention.<br>**Amount Threshold:** Based on the selected currency, enter a specific amount here. If set, when a reward amount exceeds this figure, it is forced to manual approval.<br>*\*Note: Entering 0 or leaving it blank may mean all amounts require manual approval (depends on underlying logic; testing during initial setup is recommended).* |
| **4.** | **Sorting** | |
| 4-1. | Deposit Page Sorting | Sorting configuration transitions (refer to the next page). |
| **5.** | **Cash Reward** | |
| 5-1. | Issuance Mode | Choose one. This affects the reward type for trigger conditions. |
| 5-2. | Value Setting | Choose one. This affects the reward type for trigger conditions. |
| **6.** | **Coupon Reward** | Choose between Free Bet or Live Casino coupons. |
| **7.** | **Trigger Conditions** | Different combinations based on cash and coupon rewards. Click "Add Condition" for multiple sets. |

## Tiếng Việt (VN)

Cấu hình chu kỳ tham gia (như: nạp đầu, nạp hàng ngày), kênh nạp tiền, phương thức duyệt và điều kiện kích hoạt cho phần thưởng tiền mặt hoặc phiếu thưởng.

| STT | Tên mục | Mô tả nội dung |
| :--- | :--- | :--- |
| **1.** | **Chu kỳ** | Chọn dựa trên tần suất nạp tiền (Ví dụ: Nạp mỗi lần, Nạp đầu, Nạp hàng ngày, hàng tuần). |
| 1-1. | Chọn loại nạp | Lựa chọn phân loại trên trang nạp tiền. |
| 1-2. | Quản lý loại nạp | Nhấp để mở cửa sổ bật lên (xem trang tiếp theo để biết chi tiết). |
| **2.** | **Kênh nạp tiền** | Chọn nhiều kênh được chỉ định. Mặc định là tất cả nếu không chỉ định. |
| **3.** | **Phương thức duyệt** | **Phê Duyệt Thủ Công:** Khi được chọn, tất cả đơn hợp lệ cần quản trị viên phê duyệt thủ công trước khi phát thưởng.<br>**Phê Duyệt Tự Động:** Khi được chọn, hệ thống tự động phát thưởng dựa trên quy tắc đặt trước mà không cần can thiệp.<br>**Ngưỡng Số Tiền:** Dựa trên loại tiền tệ đã chọn, hãy nhập một số tiền cụ thể vào đây. Nếu được đặt, khi tiền thưởng vượt quá số này, nó bắt buộc phải duyệt thủ công.<br>*\*Lưu ý: Nhập 0 hoặc để trống có thể có nghĩa là mọi số tiền đều cần duyệt thủ công (tùy thuộc vào hệ thống; khuyên nên kiểm tra khi thiết lập ban đầu).* |
| **4.** | **Sắp xếp** | |
| 4-1. | Sắp xếp trang nạp | Thiết lập thứ tự sắp xếp trên trang nạp tiền (xem trang tiếp theo). |
| **5.** | **Thưởng tiền mặt** | |
| 5-1. | Chế độ phát | Chọn một. Tùy chọn này ảnh hưởng đến loại thưởng của điều kiện kích hoạt. |
| 5-2. | Thiết lập giá trị | Chọn một. Tùy chọn này ảnh hưởng đến loại thưởng của điều kiện kích hoạt. |
| **6.** | **Thưởng phiếu** | Chọn giữa Phiếu cược miễn phí hoặc Phiếu cược Live Casino. |
| **7.** | **Điều kiện kích hoạt** | Các tổ hợp khác nhau dựa trên thiết lập thưởng tiền và thưởng phiếu. Nhấp "Thêm điều kiện" để tạo nhiều nhóm. |
